git init (1 раз, создает папку .git)
git add . (отслеживать все файлы)
git commit -m "comment" (Запомнить текущее состояние сайта и добавить коммент)
git push origin название ветки (git push origin master) - закачать файлы на удаленный репозиторий (github)

git branch branchName - создать новую ветку
git checkout branchName - сменить ветку
